IT BA Core Program – Module Structure

Module 1 – IT Business Analysis Foundations

  • Role and responsibilities of IT BA
  • Stakeholder analysis
  • Types and levels of requirements

Module 2 – Business Processes & BPMN

  • Identifying business processes
  • BPMN basics
  • Sample case from banking

Module 3 – Requirements Engineering & Documentation

  • BRD, user stories, use cases
  • Defining acceptance criteria
  • Managing change requests

Module 4 – IT BA in an Agile Environment

  • Product backlog, sprint planning, review
  • IT BA activities inside the sprint
  • Breaking down and prioritizing user stories

Module 5 – Technical Basics: APIs, Data, SQL

  • What is an API (REST, SOAP)
  • Data modeling concepts
  • Simple SQL queries

Module 6 – Final Project & Portfolio

  • Selecting a real business scenario
  • Building the end-to-end documentation package
  • Final demo and feedback
