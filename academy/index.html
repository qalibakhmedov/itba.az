Niyə itba.az?

Akademiya yalnız IT Biznes Analitikasına fokuslanır; bank, fintech və korporativ mühitlərdə real layihələrə hazır olmanız üçün bacarıq əsaslı, praktika yönümlü təlim təqdim edir.

Yalnız IT Biznes Analitikasına fokuslanma

Bütün modullar (biznes prosesləri, tələblər, Agile, API-lər, data) IT BA rollarının gündəlik ehtiyaclarına uyğunlaşdırılıb.

Bacarıq əsaslı yanaşma

Təlim, bacarıqları sistemli şəkildə ölçmək və inkişaf etdirmək üçün IT BA bacarıq matrisi ətrafında qurulub.

Biznes dəyərinə yönəlmiş

Hər modul yalnız nəzəri bilik deyil, layihələrdə real biznes dəyəri yaratmaq üçün nəzərdə tutulub.

Bu proqram kimlər üçündür?

Bu proqram IT Biznes Analitikasına keçmək istəyən karyera dəyişdirənlər, həmçinin daxili BA bacarıqlarını gücləndirmək istəyən korporativ komandalar üçün nəzərdə tutulub.

Karyera dəyişdirənlər

Əməliyyatlar, müştəri xidməti, çağrı mərkəzləri, biznes rollarından IT BA-ya keçmək istəyən insanlar.

Junior / gələcək analitiklər

Struktur yol və daha güclü bünövrə axtaran, müəyyən IT və ya BA təcrübəsi olan insanlar.

Məhsul / Layihə komandaları

Tələb keyfiyyətini və developer komandaları ilə kommunikasiyanı yaxşılaşdırmaq istəyən Product Owner-lər, PM-lər.

Banklar və fintech şirkətləri

Daxili IT BA bacarıqlarını qurmaq və ya gücləndirmək istəyən təşkilatlar (mobil bankçılıq, kartlar, ödənişlər, daxili sistemlər və s.)

IT BA Əsas Proqramı – Modul Strukturu

Modul 1 – IT Biznes Analitikası Əsasları

  • IT BA-nın rolu və məsuliyyətləri
  • Maraqlı tərəflərin analizi
  • Tələblərin növləri və səviyyələri

Modul 2 – Biznes Prosesləri və BPMN

  • Biznes proseslərinin müəyyən edilməsi
  • BPMN əsasları
  • Bank sahəsindən nümunə keys

Modul 3 – Tələblərin Mühəndisliyi və Sənədləşmə

  • BRD, user story-lər, use case-lər
  • Qəbul meyarlarının müəyyən edilməsi
  • Dəyişiklik sorğularının idarə edilməsi

Modul 4 – Agile Mühitdə IT BA

  • Product backlog, sprint planlaması, review
  • Sprint daxilində IT BA fəaliyyətləri
  • User story-lərin bölünməsi və prioritetləşdirilməsi

Modul 5 – Texniki Əsaslar: API-lər, Data, SQL

  • API nədir (REST, SOAP)
  • Data modelləşdirmə konsepsiyaları
  • Sadə SQL sorğuları

Modul 6 – Yekun Layihə və Portfolio

  • Real biznes ssenarisinin seçilməsi
  • Uçdan-uca sənədləşmə paketinin hazırlanması
  • Yekun demo və rəy

Tez-tez verilən suallar

Maksimum çeviklik üçün əsasən onlayn/hibrid formatdadır. Kifayət qədər tələb olduqda Bakıda oflayn qruplar da təşkil olunur.
Bəli, proqram xüsusilə biznes tərəfindən IT BA-ya keçid edənlər üçün nəzərdə tutulub. Əsaslardan başlayıb sistemli şəkildə irəliləyirik.
İş təminatı vermirik, lakin şanslarınızı artırmaq üçün portfolio hazırlığı, CV/LinkedIn optimallaşdırması və müsahibə hazırlığında sizə dəstək oluruq.
Ödəniş birdəfəlik və ya hissə-hissə edilə bilər. Ətraflı qiymət və ödəniş planları müraciət formasını göndərdikdən sonra sizinlə paylaşılır.
Bəli, korporativ proqramlar tam fərdiləşdirilə bilər. Kurikulumu təşkilatınızın real daxili keysləri əsasında uyğunlaşdırırıq ki, maksimum aktuallıq və effekt təmin olunsun.
